sql_subscript(*sub)

Return a Subscript with the given arguments, representing an SQL array access.

Sequel[:array].sql_subscript(1) # array[1]
Sequel[:array].sql_subscript(1, 2) # array[1, 2]
Sequel[:array].sql_subscript([1, 2]) # array[1, 2]
Sequel[:array].sql_subscript(1..2) # array[1:2]
Sequel[:array].sql_subscript(1...3) # array[1:2]

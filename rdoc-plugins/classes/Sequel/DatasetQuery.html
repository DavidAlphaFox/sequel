query(&block)

Translates a query block into a dataset. Query blocks are an alternative to Sequelâ€™s usual method chaining, by using instance_exec with a proxy object:

dataset = DB[:items].query do
  select :x, :y, :z
  where{(x > 1) & (y > 2)}
  reverse :z
end

Which is the same as:

dataset = DB[:items].select(:x, :y, :z).where{(x > 1) & (y > 2)}.reverse(:z)

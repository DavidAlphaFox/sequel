class Sequel::TimestampMigrator

  1. lib/sequel/extensions/migration.rb
Superclass: Migrator

The migrator used if any migration file version is greater than 20000101. Stores filenames of migration files, and can figure out which migrations have not been applied and apply them, even if earlier migrations are added after later migrations. If you plan to do that, the responsibility is on you to make sure the migrations donâ€™t conflict. Part of the migration extension.
